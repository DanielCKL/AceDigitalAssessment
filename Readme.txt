1. To set up the venv on your computer, type: 
pip -m venv .venv 
within the same folder as the 'data' folder and all the other .py files

2. To activate if you are on a windows device, type: 
activate

3. To install dependencies, type:
pip install -r pipfreeze.txt

4. To run, either click on RunQ1.bat, RunQ2.bat, RunQ3.bat, or directly use the venv's python installation to run Q1.py, Q2.py, and Q3.py

5. Answers are in Q1.docx,Q2.docx, Q3.docx, and code is in Q1.py, Q2.py, and Q3.py.
